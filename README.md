## Installation

```bash
$ docker-compose up -d
$ npm install
```

## Running the application

```bash
# development
$ npm run start

# development in watch mode
$ npm run start:dev

# production
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
